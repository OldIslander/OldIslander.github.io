Who's Who

• Created an angular.js app that allows players to guess the title of a playing track
• Implemented the game by utilizing Spotify's search API to generate an array of tracks which were randomly shuffled
• Game features several config settings, a scoring system, and a leaderboard that persists through multiple game sessions as per business requirements
• Project was a primarily solo effort; planning was done using a figma wireframe
Environment: Typescript, HTML5, CSS, angular.js, howler.js, Visual Studio Code

FastTrack Wiki

• Created a website that interfaced with a PostgresSQL repository to display employees, announcements, and teams of respective companies.
• Users could log in to view announcements; those with admin privileges could view users for each company, edit and create teams, add and remove users.
• Team collaboration: two members were responsible for the frontend functionality of the website, while another two(including myself) were responsible for developing the API that the frontend team could use to populate the website.
• Team member roles were fluid; members would shift between backend and frontend work as needed and appropriate.
Environment: Eclipse, Visual Studio Code, Spring Boot, Angular.js, Java, Javascript, git
